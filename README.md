# FTP File Transfer Monitoring & Threat Hunting Using Splunk

This repository documents an FTP threat hunting project using Splunk and Zeek FTP logs.

The project focused on monitoring FTP activity, analyzing file transfer operations, investigating user activity, and identifying potentially suspicious interactions with FTP servers.

---

## Technologies Used

- Splunk
- Zeek
- Kali Linux
- VirtualBox

---

## Focus Areas

- FTP Monitoring
- Threat Hunting
- Detection Engineering
- Network Traffic Analysis
- File Transfer Monitoring
- SOC Operations
